Installing pyproject-examples
==================

Installing `pyproject-examples` from the main channel can be achieved by:

```
conda install pyproject-examples
```

It is possible to list all of the versions of `pyproject-examples` available on your platform with `conda`:

```
conda search pyproject-examples
```
